# Qt 字体样式设置工具
✨ 基于 Qt C++ 开发的可视化字体样式预览工具

## 📝 项目介绍
本项目可以实时预览**字体、字号、粗体、斜体、上划线、下划线、删除线**效果，

## 🎯 实现功能
- ✅ 字体选择（华文隶书等）
- ✅ 字号调节（20 - 40）
- ✅ 粗体 / 斜体样式切换
- ✅ 上划线 / 下划线 / 删除线 切换
- ✅ 文字样式实时预览

## 🛠 开发环境
- **开发框架**：Qt 5 / Qt 6
- **开发语言**：C++
- **开发工具**：Qt Creator

## 使用方法
1. 选择字体
2. 选择字号
3. 勾选粗体/斜体
4. 选择线条样式
5. 实时查看预览效果

## 📁 核心文件
- `widget.h` / `widget.cpp` 主界面逻辑
- `ui_widget.h` UI 布局
- `main.cpp` 程序入口
